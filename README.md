# WebRTC Client

demo for webrtc

## how to use

### run SignalingChannel

```
git clone https://github.com/JarvisChu/SignalingChannel.git
cd SignalingChannel
./run.sh
```

### run WebRTCClient

**using chome**

```
cd electron-app
open index.html in chrome
```

**using electron**

```
cd electron-app && npm install && npm start
```
